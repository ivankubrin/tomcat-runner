Плагин tomcat-runner подходит только для многомодульной сборки.
Порядок создания проекта:
1. Установить gradle на компьютер - см. документацию (https://gradle.org/install/)
2. Создать директорию, которая будет являться проектом (в текущем примере - tomcat-runner-app).
3. В терминале выбрать данную директорию и выполнить команду "> gradle init".
4. Создать проект с параметрами:
    - тип - 2: application;
    - язык - 5: Scala;
    - разделение фенкциональности по подпроектами - 1: no;
    - язык для скриптов сборки - 1: Groovy;
    - название проекта - по умолчанию;
    - пакет с исходным кодом - по умолчанию.
5. Открыть проект в Intellij IDEA и создать в корне проекта файл build.gradle.
